# Define

Understand user needs and develop a problem hypothesis.

# Summary

In this phase, the results from the Empathise phase are interpreted and analyzed. The user needs and findings are prepared for the further process, e.g. in the form of personas or an Emotional Journey Map. The mode here is convergent, i.e., the insights are distilled. If the define phase falls short, there is a risk that simply random ideas are collected or the user-centeredness is lost (design for "myself").

# Result

A problem hypothesis that creates a clear task for idea collection in the next phase.

# Questions to be answered in this phase

  • What are the real needs and problems of the users?
  • Where should we start looking for a solution?
